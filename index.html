Welcome! Here you can find installation instructions for Debian/Ubuntu and RHEL/CentOS/Fedora systems.

Debian / Ubuntu (APT)

Add this repository to your system:

echo "deb [arch=amd64] https://.github.io//apt-repo stable main" | sudo tee /etc/apt/sources.list.d/myrepo.list
sudo apt update
sudo apt install <package-name>

RHEL / CentOS / Fedora (YUM/DNF)

Create a new repo file:

sudo tee /etc/yum.repos.d/myrepo.repo <<EOF
[myrepo]
name=My Custom Repo
baseurl=https://.github.io//rpm
enabled=1
gpgcheck=0
EOF

sudo dnf install <package-name>
