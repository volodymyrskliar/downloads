Debian / Ubuntu (APT)

Add this repository to your system:

curl -fsSL https://volodymyrskliar.github.io/downloads/public.key | sudo gpg --dearmor -o /usr/share/keyrings/scalr.gpg
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/scalr.gpg] https://volodymyrskliar.github.io/downloads/apt-repo stable main" | sudo tee /etc/apt/sources.list.d/scalr.list
sudo apt update
sudo apt install scalr-agent
